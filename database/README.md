# SP MAPI — Database Schemas

## Tech Options
- **MongoDB** — Mongoose schemas
- **PostgreSQL** — Sequelize models (alternative)

## Core Collections / Tables
1. **users** — Customers (OTP login)
2. **properties** — Plot listings
3. **bookings** — Plot booking records
4. **service_requests** — Land measurement requests
5. **amins** — Field surveyors
6. **admins** — District administrators
7. **payments** — Razorpay payment records
8. **documents** — Uploaded KYC/land documents
9. **audit_logs** — Admin action logs
10. **districts** — District master data
