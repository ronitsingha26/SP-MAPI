# SP MAPI — Backend (Node.js + Express)

This folder will contain the backend REST API.

## Planned Structure
- `server.js` — Express server entry point
- `controllers/` — Route handlers
- `routes/` — API route definitions
- `models/` — Mongoose / Sequelize models
- `middleware/` — Auth, RBAC, error handling
- `utils/` — OTP, PDF generation, notifications
- `config/` — DB config, env variables

## Tech Stack
- Node.js + Express.js
- MongoDB (Mongoose) or PostgreSQL (Sequelize)
- JWT Authentication
- Razorpay Payment Gateway
- Multer (File Upload)
- Nodemailer + SMS Gateway (Twilio/MSG91)
